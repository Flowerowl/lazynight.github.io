1.

git fetch origin master:tmp

git diff tmp

git merge tmp

git branch -d tmp

从远程获取最新的版本到本地的tmp分支上

之后再进行比较合并

2.

git pull：相当于是从远程获取最新版本并merge到本地

git pull origin master

上述命令其实相当于git fetch 和 git merge

在实际使用中，git fetch更安全一些

因为在merge前，我们可以查看更新情况，然后再决定是否合并
